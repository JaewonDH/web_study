CSS 선택자

  1. #id 선택자
    2. #id { border: 1px solid; }
  2. .class 선택자
  3. input[type=text] 속성 선택자
  4. 자손 선택자 div > p div 바로 하위에 있는 p태그를 선택
  5. 후손 선택자 div li div 하위에 있는 모든 li를 선택
  6. 동위 선택자
    • div + p div와 동일한 위치에서 바로 다음 태그가 p인 태그를 선택
    • div ~ p div 동일한 위치에서 첫번째 div 밑에 있는 모든 p태그를 선택
  7. 반응 선택자

    8. li 태그에 마우스가 over되면 선택된 스타일이 적용 된다.`

    #li:hover { border: 1px solid; }
